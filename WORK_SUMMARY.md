# 环境控制项目工作整理文档

## 1. 项目概览
本项目为基于 **LVGL + Linux framebuffer** 的嵌入式环境控制系统，目标平台分辨率为 **1024x600**，包含：
- 传感器采集（AHT20 温湿度、BH1750 光照）
- 风扇控制（fan0/fan1 自适应）
- RGB 灯光控制（GPIO117/116/109）
- GUI 多页面（首页、详情、模式管理）
- 历史数据记录（内存环形缓存 + CSV 落盘）

---

## 2. 工作目录内容（结构化整理）

### 2.1 业务代码（核心）
- `src/main.c`：程序入口、LVGL 主循环。
- `gui/custom.c`：主界面与业务逻辑核心（页面构建、刷新、事件、模式策略、传感器数据落地到 UI）。
- `gui/custom.h`：GUI 自定义接口。
- `gui/setup_scr_screen.c`：屏幕构建入口。
- `gui/events_init.c/.h`：事件初始化（当前较轻量）。

### 2.2 设备与硬件抽象
- `sensor/i2c_bus.c/.h`：I2C 总线操作。
- `sensor/aht20.c/.h`：AHT20 温湿度采集。
- `sensor/bh1750.c/.h`：BH1750 光照采集。
- `device/fan.c`：风扇控制，支持 fan0/fan1 自适应。
- `device/gpio.c`：GPIO 导出、方向、读写。
- `device/led.c`：基础 LED 设备接口（历史兼容）。
- `include/board.h`：硬件引脚定义（FAN GPIO、RGB GPIO）。
- `include/fan.h / gpio.h / led.h`：设备头文件。

### 2.3 图形与字体资源
- `gui/images/*`：图标资源及 C 数组。
- `gui/guider_fonts/lv_font_kaiti_16.c`：统一中文字体资源。

### 2.4 构建与运行
- `Makefile`：项目构建脚本。
- `lv_conf.h / lv_drv_conf.h`：LVGL 与驱动配置。
- `boot/*`：开机启动相关脚本与 service 文件。

### 2.5 第三方框架
- `lvgl/*`：LVGL 源码。
- `lv_drivers/*`：LVGL 驱动层。

> 说明：工作目录完整文件数量较大（LVGL 全量源码）。如需完整逐文件清单，可执行：
> `rg --files`

## 4. 当前模式策略（已实现）
- **手动模式**：用户直接控制风扇与 RGB，策略不覆盖。
- **自动模式**：按温湿度与光照进行联动控制。
- **节能模式**：提高触发阈值，减少动作次数，优先低功耗。
- **夜间模式**：风扇策略更保守，灯光偏低干扰。
- **均衡模式**：多条件折中决策。

---

## 5. 历史数据与记录
- 内存中维护环形历史数据（用于图表和详情窗口）。
- 每分钟写入一条记录（含温湿度、光照、风扇、RGB、模式）。
- CSV 日志文件路径：`/tmp/env_sensor_log.csv`。
